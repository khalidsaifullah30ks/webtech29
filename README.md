# Student Course Hub Website

## Overview
This is a web application developed as part of the Web Application Development module.

The system allows students to:
- View programmes (BSc and MSc)
- See programme details and modules
- Register interest in programmes
- Withdraw interest

Admins can:
- Manage programmes and modules
- Publish/unpublish programmes
- View student interest
